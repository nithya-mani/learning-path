# Learning Path Dashboard

A free, self-hosted interactive learning tracker. No backend, no accounts — just one HTML file.

## ✨ Features
- **Setup wizard** — enter your name, date range, hours goal and priority tracks on first load
- **Course tracker** — add, edit and delete courses with checkbox completion, LinkedIn toggle and date picker
- **KPI cards** — real-time progress across hours, courses, LinkedIn posts and your priority tracks
- **Charts** — category breakdown and week-by-week view
- **CSV export** — download completed courses for your master spreadsheet
- **Download your version** — generates a personalised HTML file pre-loaded with your data

## 🚀 Host on GitHub Pages (5 minutes)

1. Create a new **public** repository on GitHub (e.g. `learning-path`)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages → Source: main branch → Save**
4. Your live URL: `https://yourusername.github.io/learning-path`
5. Share the link on LinkedIn!

## 🔁 Share as a template
Anyone who opens your GitHub Pages link will see the **setup wizard** and build their own version. Their data is saved in *their* browser — completely separate from yours.

They can also click **Download My Version** to get their own personalised copy.

## 📁 Files
- `index.html` — the entire dashboard (single file, no dependencies except Chart.js CDN)
- `README.md` — this file

## 🔒 Privacy
All data is stored in **localStorage** in the user's browser. Nothing is sent to any server.
